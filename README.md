# 📱 Aurora Dossier

Aplicación móvil inteligente para la gestión administrativa y operativa empresarial, potenciada con Inteligencia Artificial.

# 📌 Descripción del Proyecto

Aurora Dossier es una aplicación móvil diseñada para optimizar la gestión de operaciones empresariales y personales mediante el uso de Inteligencia Artificial.

El sistema elimina la carga administrativa manual y mejora la toma de decisiones mediante:

- 📊 Supervisión de métricas empresariales

- 🧾 Gestión estructurada de informes técnicos

- 🤖 Automatización inteligente con IA

- 📈 Visualización de indicadores clave

---

# 🎯 Objetivo

Desarrollar una aplicación móvil integral que permita:

- Centralizar la creación y gestión de informes.

- Optimizar la clasificación y priorización de tareas.

- Generar alertas predictivas mediante análisis inteligente.

- Digitalizar documentos físicos mediante OCR.

- Mejorar la experiencia de usuario con diseño moderno y accesible.

---

# 🧠 Módulos Principales
🔹 Módulo Administrativo

- Gestión de usuarios y roles.

- Supervisión de métricas.

- Dashboard con gráficos.

- Publicación de novedades internas.

🔹 Módulo Operativo

- Creación y edición de informes.

- Bandeja de tareas pendientes.

- Gestión de estados (Borrador, Enviado, Revisado, Archivado).

- Historial con buscador y calendario.


# 🤖 Funcionalidades con Inteligencia Artificial

- ✍️ Asistente de redacción inteligente

- 🚨 Análisis de gravedad y clasificación automática

- 📄 Extracción OCR desde documentos físicos

- 📊 Generación automática de reportes detallados
